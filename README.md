# 一番賞抽獎券

A Japanese-style Ichiban Kuji (一番賞) lottery ticket web app. Customize prizes, draw tickets with a satisfying tear animation, and reveal your reward.

## Features

- Configurable prizes with ranks, names, colors, and quantities
- Tear-open ticket animation
- Persistent state via localStorage (draws survive page refresh)
- Mobile-friendly, responsive layout
- No dependencies — pure HTML, CSS, and JavaScript

## Usage

1. Tap ⚙️ to configure the event title, prizes, and ticket text
2. Tap any ticket to draw it
3. Drag upward to tear the ticket open and reveal your prize
4. Tap **重置抽獎** to reset and start a new round

## Deploy
